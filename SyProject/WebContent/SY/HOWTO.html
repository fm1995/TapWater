Advanced Tables

To create an advanced table using the Data Table Plugin add the following attribute to any table

<table class="datatable">

-- OR --

In your javascript grab the table element you wish to use and attach the data table plugin like below:

$('#myDataTable').dataTable ();

Grid Layout

The layout is based on a 12 column grid. Any combination of columns can be used in a row so long as the sum of the columns is not greater than 12. Columns are created by adding one of the classes .x1-.x12 to a div.portlet.

Here are a few examples of a correct column layout. 

<div class="x12">col</div>
			
<div class="x6">col</div>
<div class="x6">col</div>

<div class="x8">col</div>
<div class="x4">col</div>

<div class="x4">col</div>
<div class="x4">col</div>
<div class="x4">col</div>

Here is an example of an incorrect column layout. As you'll see, the columns add up to 14 instead of 12.

<div class="x8">col</div>
<div class="x6">col</div>

Forms


<form class="form" action="" method="get" accept-charset="utf-8">

All styled forms need to have the class .form

To create a form with the labels on top of the fields, add the class .label-top to the form tag. The default for forms is labels to the left of fields and no classes are necessary for this to work.


<div class="field">
	<label for="subject">Subject</label>
	<input type="text" name="subject" value="" id="subject" class="medium" />				
	<span class="field_help">This is the help text for the field</span>
	<span class="field_error">This field is invalid</span>
</div> <!-- .field -->

Specify the label position using the class .label-top to create labels that are on top of the fields. Use the class .label-inline to create labels that are to the left of the fields. These two classes go on the main form tag.

Input elements can contain the classes .small, .medium, .large to specify the size of the an input field.

Specify field help text by adding SPAN.help_text

Uniform Plugin

To utilize the uniform plugin simply add the following class to your form tag: .uniform

<form class="form uniform">

Charting Helper

The quickest and easiest way to create a chart is to add the class .stats and a data-chart attribute with chart type (line, area, bar, pie).


<table class="stats" data-chart="line">

-- OR --

Attach the jquery visualize plugin to the table you want in your javascript file.


$('#myChartTable').visualize({ type: 'line' });

Simply modify the paragraph tag contained in .apg-features container. Be careful not to enter too much text for this.

Tabs

Below is the container for the tab widget. All panels must go in here.

Below is the example code for tab navigation. The href attribute for each tab must be set to the id of the content that it shows. For example, if you have a div with the id of "tab4", then the tab attribute needs to be #tab4.

    <li><a href="#tab3">Tab #3</a></li>
